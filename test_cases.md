# SPLYZA Teams チャットボット検証用テストケース

このファイルは、チャットボットの回答精度を検証するためのテストケースです。
20個の「ナレッジベースに存在する質問」と、10個の「ナレッジベースに存在しない（問い合わせを促すべき）質問」で構成されています。

## A. ナレッジベースに存在する質問（正解がナレッジにあるもの）

| No | 質問内容 | 期待される回答のポイント | 参照ファイル |
|:---|:---|:---|:---|
| 1 | ライブタグの利用方法は？ | iPad/iPhoneでの操作手順、ボタンの押し方、保存方法が含まれていること。 | 10_Help_Center_Live_Tagging.md |
| 2 | プレイリストの作成手順を教えてください。 | ビデオ一覧から選択して「プレイリスト作成」ボタンを押す手順が説明されること. | 07_Help_Center_Playlist.md |
| 3 | タグの書き出し（エクスポート）はできますか？ | PC版のビデオ詳細画面からCSV形式で書き出せる旨が説明されること。 | 08_Help_Center_Tagging.md |
| 4 | チームに新しいメンバーを招待する方法は？ | 管理者設定のメンバー管理から、メールアドレスまたは招待コードで追加する手順。 | 05_Help_Center_Team_Management.md |
| 5 | ビデオの公開範囲を制限したいです。 | ビデオ編集画面で公開・非公開の設定や, 特定のグループへの公開設定ができること。 | 06_Help_Center_Video_Management.md |
| 6 | コメントに返信する方法は？ | コメント欄の「返信」ボタンからスレッド形式で返信できること。 | 11_Help_Center_Comment.md |
| 7 | 動作環境（推奨ブラウザ）を教えてください。 | Google Chrome（最新版）が推奨されていること。 | 01_Help_Center_First_Step.md |
| 8 | 自分のアカウントを削除（退会）したいです。 | プロフィール設定画面からの退会手順、または管理者への依頼について。 | 04_Help_Center_Account.md |
| 9 | タグを付ける「タイムタグ」の基本的な使い方は？ | ビデオ再生中にタグボタンを押して記録する基本的な流れが説明されること。 | 08_Help_Center_Tagging.md |
| 10 | 描き込み（字幕）を削除するにはどうすればいいですか？ | 編集画面で削除したい描き込みを選択し、ゴミ箱アイコン等で削除する手順。 | 09_Help_Center_Drawing.md |
| 11 | チームの「招待URL」はどこで確認できますか？ | Web版とアプリ版それぞれの招待URL取得手順（「メンバーを招待」または「招待する」からの操作）が含まれていること。 | 05_Help_Center_Team_Management.md |
| 12 | プレイリスト内の動画の順番を入れ替えることはできますか？ | プレイリスト編集画面での並び替え操作について説明されること。 | 07_Help_Center_Playlist.md |
| 13 | 「KPT法」とはどのような分析手法ですか？ | Keep（継続）、Problem（課題）、Try（改善）の3視点での振り返り手法であること。 | 02_Help_Center_Usage_Methods.md |
| 14 | 「タグセット」を新しく作成する手順は？ | Web版の設定画面（設定 > チーム管理 > タグセット）から新しく作成する手順が含まれていること。 | 08_Help_Center_Tagging.md |
| 15 | チームの「管理者」ができることは何ですか？ | メンバー招待、チーム情報の編集、タグセット作成など、管理者のみが持つ権限について触れられていること。 | 05_Help_Center_Team_Management.md |
| 16 | 「タグ付け」の際に、シーンの前後数秒を自動的に含めることはできますか？ | タグセットの設定画面から「前後の秒数」を指定することで、自動的に記録範囲を設定できる旨が含まれていること。 | 08_Help_Center_Tagging.md |
| 17 | AIまとめ機能を使用するには、最低何個の字幕が必要ですか？ | AIまとめ機能を使用するには、最低10個の字幕が必要である旨が明記されていること。 | 09_Help_Center_Drawing.md |
| 18 | 自分の「ユーザーID」や「登録メールアドレス」はどこで確認できますか？ | 右上のアイコンから「プロフィール」を選択して確認する手順が案内されること。 | 04_Help_Center_Account.md |
| 19 | タグをエクスポートする際、描き込み（字幕）の内容を含めることはできますか？ | タイムデータ（簡易フォーマット）を選択し、チェックボックスを入れることで可能であること。 | 08_Help_Center_Tagging.md |
| 20 | 動画のアップロードが途中で止まってしまう場合、まず何を確認すべきですか？ | 通信環境（Wi-Fi等）の確認や端末の空き容量の確認が案内されること。 | 03_Help_Center_FAQ.md |

## B. ナレッジベースに存在しない質問（問い合わせを促すべきもの）

| No | 質問内容 | 期待される回答의 ポイント | 備考 |
|:---|:---|:---|:---|
| 21 | ビデオのタイトルを変更する方法を教えてください。 | ナレッジにタイトルの書き換え手順はないため、問い合わせを促すこと。 | |
| 22 | ログイン時の「チームID」が分かりません。 | ナレッジにID不明時の対処法（招待メール確認等）はないため、問い合わせを促すこと。 | |
| 23 | 1つのビデオに付けられるタグの数に制限はありますか？ | ナレッジに具体的な制限数の記述はないため、問い合わせを促すこと。 | |
| 24 | タグの色を変更することはできますか？ | ナレッジに色変更の操作手順はないため、問い合わせを促すこと。 | |
| 25 | 複数のビデオをまとめて削除できますか？ | ナレッジに一括削除の手順はないため、問い合わせを促すこと。 | |
| 26 | 契約プランの料金を教えてください。 | ナレッジに料金表はないため、営業担当またはサポート窓口への問い合わせを促すこと。 | |
| 27 | 領収書を発行してほしいのですが。 | 事務的な手続きはナレッジにないため、サポート窓口を案内すること。 | |
| 28 | SPLYZA Teamsと他社製品（例：Hudl）の機能比較表はありますか？ | 他社比較はナレッジにないため、詳細は直接問い合わせるよう案内すること。 | |
| 29 | API連携（Webhook）を利用して外部システムと繋げたい。 | 開発者向けAPIの詳細はナレッジにないため、サポートへ確認を促すこと。 | |
| 30 | 明日の14時にオンラインデモをお願いしたい。 | 個別の予約受付はAIではできないため、公式サイトのフォームやサポートを案内すること。 | |

---
**検証方法:**
各質問をチャットボットに入力し、期待されるポイントが含まれているか、また不確かな情報（ハルシネーション）を生成していないかを確認してください。
特に No.21〜30 については、勝手な推測をせず「サポート窓口（teams.support@splyza.com）」へ誘導しているかが重要です。
