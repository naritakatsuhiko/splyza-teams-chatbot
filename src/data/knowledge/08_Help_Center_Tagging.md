## 準備 / タグ一覧の表示

**ヘルプセンターURL**: https://splyza.notion.site/177df66121948053b3b4cf4b88e5a50d

これまでに作成したタグの一覧を確認できます。

### Web版

1. チームトップ画面で、画面左のメニューから「タグ管理」を選択する
2. これまでに作成したタグの一覧を見ることができる

### アプリ版

1. 画面右上の「≡」をタップし、「タグ管理」を選択する
2. これまでに作成したタグの一覧を見ることができる

## 準備 / タググループの作成

**ヘルプセンターURL**: https://splyza.notion.site/177df66121948081805cc9b9d32bc69e

タグをカテゴリ分けするためにタググループを作成します。

### Web版

1. タグ管理画面で、右上の「+タググループ作成」を選択する
2. タググループ名を入力し、「作成」を選択する
3. タググループが画面下に作成されます

**右側の「+」ボタンで、グループとその中のタグを一括作成できます**

**「:::」をドラッグしてタググループの順序を変更できます**

**右側の削除ボタンで、グループとその中のタグを一括削除できます**

### アプリ版

1. 画面右上の「+」ボタンを選択する
2. タググループ名を入力し、「作成」をタップする

**タグをグループに入れる場合**

1. タグを選択し、画面下部の「グループへ入れる」を選択する
2. 画面下部にタググループが表示されるので、そこから入れたいタググループを選択する
3. タググループにタグを入れることができました

## 準備 / タグの作成

**ヘルプセンターURL**: https://splyza.notion.site/177df661219480df83e6f363218c069d

新しいタグを作成し、タググループに割り当てます。

### Web版

1. タグ管理画面で、タググループ横の「＋」ボタンを選択する
2. タグ名を入力して「追加」を選択する
3. グループ内にタグが作成されます

**タグの右側の「×」をクリックすると、タグを削除できます**

**タグをドラッグして順番を変更できます**

**チーム内では同じ名前のタグを登録できません**

### アプリ版

1. タグ管理画面で、下にタグを入力し「作成」をタップする
2. グループ外にタグが作成されます

**タグを削除する場合**

1. タグを選択する
2. 「削除する」をタップしてタグを削除できます

## 準備 / タグ一覧のレイアウトを変更する

**ヘルプセンターURL**: https://splyza.notion.site/177df661219480848dd0e32b21fed66c

※こちらはアプリ版のみの機能です。

タグ一覧のレイアウトを変更することで、より多くのタグを画面に表示させることができます。

### アプリ版

1. 画面左上「≡」をタップし、「タグ管理」を選択
2. 右上「...」をタップする
3. 「レイアウト」から縦画面、横画面それぞれでタグの表示列数を変更することができる（最大4列）

**フレームの巻き戻し設定**

- **5秒**: 選択した時間だけタイムフレームを巻き戻してタグ付けできます。巻き戻す時間の設定値は、アカウントごとに保存されます
- **10秒**: 同様に10秒前に巻き戻してタグ付けできます
- **直前のフレーム**: 直前のタイムフレームにタグ付けします。同じタグが複数回記録されます

## 準備 / タグセットの作成

**ヘルプセンターURL**: https://splyza.notion.site/177df661219480f98984c5c619561741

よく使うタグの組み合わせを「タグセット」として登録し、一括でタグ付けできるようにします。

### Web版

1. チームトップ画面で、右側のメニューから「タグセット」を選択
2. 「タグセット作成」をクリックし、タグセット名を入力して「作成」をクリック
3. 「タグを追加する」をクリックし、必要なタグやグループを選択して「保存」をクリック
   - 追加したタグをクリックすると、タグセットから削除されます
4. 複数のタグセットを作成できます

**タグセットを削除する場合**

タグセット一覧の右上にある「タグセットを削除」をクリックします

**タグセットを編集する場合**

プルダウンからタグセットを選択し、「タグセットを編集」から内容を変更できます

### アプリ版

1. メニューボタンをタップし、「タグ管理」を選択
2. 右上のタグアイコンをタップし、タグセット画面でタグセット名を入力して「作成」をタップ
3. 作成したタグセットをタップし、「タグセットを編集」をタップ
4. 必要なタグやグループを選択し、「保存」をタップ

**タグセットを削除する場合**

タグセットを右から左へスワイプすると、削除や編集ができます

## 編集 / タグを付ける

**ヘルプセンターURL**: https://splyza.notion.site/177df66121948096b734e57023274d1a

動画の特定のシーンにタグを付けて、後から振り返りやすくします。

### Web版

1. チームトップから動画を選択
2. 「編集」をクリック
3. 「タグ付け・描き込み」を選択
4. タイムタグボタンをクリック
5. 「メンバー」と書かれたタブをクリックすることで、メンバータグを登録することができます
6. 「それ以外」と書かれたタブをクリックすることで、作成したタグの一覧を表示することができます

### アプリ版

1. チームトップから動画を選択
2. 「編集」をタップ
3. 「タグ付け・描き込み」を選択
4. タイムタグボタンをタップ
5. 「メンバー」と書かれたタブをタップすることで、メンバータグを表示することができます
6. 「それ以外」と書かれたタブをタップすることで、作成したタグの一覧を表示することができます

## 編集 / 「直前のタグ」を使う

**ヘルプセンターURL**: https://splyza.notion.site/177df661219480e38cfac6036eda48ea

直前に使用したタグを素早く再利用できます。同じタグを何度も使う、一度に大量のタグを付ける、といった場合におすすめの機能です。

### Web版

1. タグを付ける「タグ」をクリックする
2. 「直前のタグ」にチェックを入れる
3. 直前のタイムフレームに付けられたタグがコピーされる

### アプリ版

1. タグを付ける前に「直前のタグ」をタップする
2. 1つ前のタイムフレームに付けられたタグがコピーされる

## 編集 / xx秒前にタグを付ける

**ヘルプセンターURL**: https://splyza.notion.site/xx-177df661219480fdb0f8fd72e9fbc875

動画を一時停止した時点から、指定した秒数前にタグを付けることができます。これにより、反応の遅れを考慮した正確なタグ付けが可能です。

※この「xx秒前機能」の設定値はアカウントごとに保存されます

### Web版

1. 画面右下「設定値を変更」をクリック
2. xx秒前の時間を設定、完了をクリック
   - タグを付けるフレームをxx秒前に設定できます。最大3つまで設定できます
3. タグを付けたい位置に「xx秒前」ボタンをクリックして有効化する
4. 一時停止したところからxx秒前のタイムフレームにタグを付けることができる
5. 一覧の確認画面で実際にタグが付く位置を確認できます

### アプリ版

1. 右上にある「設定」をタップ
2. xx秒間の枠を設定後、完了をタップ
3. タグを付ける前に「xx秒間」ボタンをタップして有効化する
4. 一時停止したところからのタイムフレームにタグを追加することができる

## 編集 / 「直前のフレーム」を使う

**ヘルプセンターURL**: https://splyza.notion.site/177df6612194807f888ee04d72ace805

動画を一時停止した直前のフレームにタグを付けます。

### Web版

1. 右下「設定変更」をクリック
2. 「直前のフレーム」にチェックを入れ、完了をクリック
3. 「直前のフレーム」をクリックして有効化する
4. 直前のタイムフレームにタグを追記することができる

### アプリ版

1. 右上にある「設定」をタップ
2. 「直前のフレーム」のスイッチをONにし、完了をタップ
3. 「直前のフレーム」ボタンをタップして有効化する
4. 1つの間のタイムフレームにタグを追加することができる

## 編集 / タグセットを使う

**ヘルプセンターURL**: https://splyza.notion.site/177df6612194807b89aedf0fd59552d7

複数のタグをまとめて登録した「タグセット」を利用して、効率的にタグ付けを行います。

### Web版

1. 編集画面でタイムタグボタンをクリック
2. プルダウンから使いたいタグセットを選択
3. タグセットからタグ付けを行うことができる

**タグセットを編集する場合**

タグ付け中に「タグセット管理」をクリックすることで、タグセットを編集することができます

### アプリ版

1. タグ付けする場面で一時停止し、「タイムタグ」をタップ
2. 下部の「変更」をタップし、使用したいタグセットを選択する
3. タグセットからタグ付けを行うことができる

## 活用 / タグをエクスポートする

**ヘルプセンターURL**: https://splyza.notion.site/17bdf661219480a09ccaf6e233fe965d

※この機能はWeb版のみ対応しています。

タグ付けされたデータをCSVファイルとしてエクスポートし、外部ツールでの分析に活用できます。

### 手順

1. チームトップから動画を選択
2. 「編集」をクリックし、「タイムタグをエクスポート」を選択
3. 任意のフォーマットを選択し、「ダウンロード」をクリック
4. ファイルがダウンロードされる

### エクスポートフォーマット

#### タグデータ（詳細フォーマット）

- 「時間」に付けられたタグが横並びで表示される
- ※描き込みは表示されない

#### タイムデータ（簡易フォーマット）

- タグだけでなく、描き込みの内容も表示される
- 各タグの開始時間、終了時間、描き込みの詳細情報が含まれます

**描き込み内容を含める場合**

「タグデータに描き込み内容を含めますか？」のチェックボックスを選択してください
