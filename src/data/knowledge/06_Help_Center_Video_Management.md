## 動画をアップロードする

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481dc8c95d2e17dcd7dfb

動画をアップロードすることで、チームメンバーに共有することができます。

### Web版

1. 左のメニューから「動画をアップロード」をクリック
2. 「ファイル選択」をクリックし、アップロードする動画を選択（※複数選択可能）
3. タイトルと詳細情報を入力し、アップロード先のルームをプルダウンから選択
4. 右上の「アップロード」をクリックしてアップロードを開始

**アップロードをキャンセルする場合**

右の「×」ボタンをクリックしてください。

**複数の動画をまとめてアップロードする場合**

右上のトグルを有効にすると、まとめてアップロードできます。

### アプリ版

1. 右下の「＋」をタップし、「動画をアップロード」を選択
2. アップロードする動画をタップし、右上の「次へ」をタップ（※複数選択可能）
3. アップロード先のルームを選択し、右上の「完了」をタップ
4. 動画ごとにタイトルと詳細情報を入力（※上部の画像を左右にスワイプして入力対象を切り替え）
5. 右上の「保存」をタップしてアップロードを開始

**アップロードをキャンセルする場合**

アップロード中の動画を右から左へスワイプしてください。

### アップロード状況の確認

アップロード状況は、通知メニューから「アップロード状況」を選択することで確認できます。

- **アップロード可能時間**: 契約によって適用されたアップロード時間の上限時間
- **残り時間**: 現在の契約期間における残りのアップロード時間
- **アップロード進捗**: 各動画のアップロード進捗率が表示されます

## ルーム新規作成

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481548027c215187a5e67

ルームを作成することで、動画を整理・分類して管理できます。

### Web版

1. 左のメニューから「ルーム」を選択
2. 「ルーム新規作成」を選択
3. アイコンと色を選択（アルバムから選択も可）
4. ルーム名（必須）とコメント（任意）を入力
5. メンバーを選択（チェックボックスで選択）
6. 「作成」をクリック

### アプリ版

1. メニューから「ルーム新規作成」を選択
2. ルーム名（必須）とコメント（任意）を入力
3. カメラアイコンを選択
4. アイコンと色を選択（アルバムからの選択も可）
5. 「完了」をタップ
6. ルームのプライバシー設定を選択
7. 「保存」をタップ

## ルームのプライバシー設定

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481d08f7cc445cd32c1d4

ルームのプライバシー設定を変更することで、動画の閲覧権限を管理できます。

### Web版

1. プライバシー設定をするルームを選択
2. 「ルーム設定」をクリック
3. 公開設定を選択
   - **プライベート**: 特定のメンバーのみに公開
   - **メンバー**: チームメンバー全員に公開
   - **公開**: すべての人に公開
4. 閲覧権限があるかチェックする
5. メンバーリストからも確認可能です

### アプリ版

1. トップ画面のルームを選択
2. プライバシー設定するルームを選択
3. 右上の「...」をタップ
4. 「ルームのプライバシーを編集する」を選択
5. 公開設定を選択
   - **特定のメンバーのみに公開**: 選択したメンバーのみに公開
   - **全員に公開**: チームメンバー全員に公開
6. メンバーを選択（複数選択可能）
7. 「完了」をタップ

## 動画をアプリ内でダウンロードする

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481cdb9aafd8a1592a55c

※この機能はアプリ版のみご利用いただけます。

アプリ内で動画をダウンロードすることで、より便利に視聴できます。

### メリット

- 通信量を節約して動画を再生できます
- 通信制限中などの不安定な環境でもスムーズに再生できます
※動画のダウンロード操作や、動画視聴以外の操作（タグ付けや描き込みなど）には通信環境が必要です。また、アプリの起動やログインには通信環境が必要な場合があります。

### 条件

- スマートフォン・タブレットに十分な空き容量があること
- Wi-Fi環境があること

### ダウンロード方法

1. ダウンロードしたい動画をタップ
2. 「ダウンロード」をタップ
3. 容量を確認し、「ダウンロード」をタップ
4. ダウンロードが完了するまでお待ちください

### ダウンロードした動画の管理

メニューボタン「≡」から「ダウンロード」を選択すると、ダウンロードした動画の一覧が表示されます。

動画を削除するには、右から左にスワイプして「削除する」をタップしてください。

**※一覧から削除しても、クラウド上の動画は削除されません。**

### 写真アプリで開く方法（iOS）

1. 画面右下の「ダウンロードアイコン」を選択
2. 「ダウンロード」を選択
3. 「開く」を選択
4. ダウンロードした動画ファイルを選択
5. アプリが起動され、動画を見ることができます

## 動画をmp4ファイルとしてダウンロードする

**ヘルプセンターURL**: https://splyza.notion.site/mp4-176df66121948190829fd5d0a330180e

SPLYZA Teamsにアップロードした動画は、mp4形式でダウンロードできます。

### 注意事項

- ※この機能はWeb版のみ対応しています
- ※「管理者」または「コーチ」権限を持つアカウントのみ、この機能を使用できます
- ※一部のプランではご利用いただけません。あらかじめご了承ください

### 手順

1. ダウンロードしたい動画を選択
2. 「編集」から「動画をダウンロード」を選択
3. 保存先を確認して「保存」をクリック
4. 動画のダウンロードが開始

### 保存先について

- 動画は通常、PCの「ダウンロード」フォルダに保存されます
- 保存先を確認するには、Chromeブラウザの3点メニューから「ダウンロード」を選択し、「フォルダを表示」（Macの場合は「Finderを表示」）をクリックしてください

## 動画を他のルームに複製する

**ヘルプセンターURL**: https://splyza.notion.site/176df66121948148b7f8effe9958b63b

動画を他のルームに複製することで、同じ動画を複数のルームで管理できます。

### Web版

1. 動画を選択
2. 右上の「...」をクリック
3. 「動画を他のルームに複製する」を選択
4. タイムタグや注記を引き継ぐか、引き継がないかを選択
5. 複製先のルームを選択
6. タイトルを編集
7. 「複製」をクリック

### アプリ版

1. 動画を選択
2. 右上の「編集」を選択
3. 「動画を他のルームへ複製する」を選択
4. タグや注記を引き継ぐか、引き継がないかを選択
5. 保存先のルームを選択
6. タイトルと詳細情報を編集
7. 「保存」をタップ

**※注意**: この操作は実行後、取り消すことができません。

## 動画を共有する

**ヘルプセンターURL**: https://splyza.notion.site/176df66121948122af45d2e63b95217d

SPLYZA Teamsにアップロードした動画は、共有することができます。SPLYZA Teamsユーザーはタグや描き込み付きで受け取ることができ、SPLYZA Teams以外のユーザーは動画ファイルとして受け取ることができます。

### 権限による共有機能の制限について

共有機能は、セキュリティを高めるため、権限によって制限されています。

- **動画を共有する**: 管理者、コーチ
- **SPLYZA Teams上で動画を受け取る**: 管理者、コーチ、プレイヤーコーチ
- **動画ファイルをダウンロードする / 受け取る（SPLYZA Teams外）**: どなたでも（SPLYZA Teamsをご利用でない方も）

### 動画を送る（共有する側）

#### Web版

1. 共有する動画を選択
2. 動画再生中の「共有」を選択
3. 上から2つ目「動画を共有する」を選択
4. 「タグや描き込みのデータも引き継ぐ」のチェックボックスを選択（任意）
5. 「URLをコピー」をクリック
6. コピーしたURLを共有相手へ送信

#### アプリ版

1. 共有する動画を選択
2. 画面右上の「共有」アイコンをタップ
3. 「動画を共有する」を選択
4. 「タグや描き込みを引き継ぐ」のスイッチをON（任意）
5. 「URLをコピー」をタップ
6. コピーしたURLを共有相手へ送信

**※注意**: 動画共有後にタグや描き込みを更新しても、共有元のチームの動画には反映されません。

### 動画を受け取る（共有される側）

#### Web版

1. 共有されたURLにアクセス
2. 動画共有ページが表示されます。下にスクロールします
3. 「SPLYZA Teamsに動画をコピー」を選択
4. ログインページでログインします
5. 動画を受け取るチームとルームを選択し、「動画を受け取る」を選択
6. 受け取り完了です。「SPLYZA Teamsで動画を見る」を選択すると受け取った動画が表示されます

#### アプリ版

1. 共有されたURLにアクセス
2. 動画共有ページが表示されます
3. 「SPLYZA Teamsに動画をコピー」を選択
4. ログインページでログインします
5. 動画を受け取るチームとルームを選択し、「動画を受け取る」を選択
6. 受け取り完了です

### 動画ファイルとして受け取る（SPLYZA Teams外）

1. 共有されたURLにアクセス
2. 動画共有ページが表示されます
3. 「動画をダウンロード」を選択
4. 注意事項に同意し、「ダウンロード」を選択
5. これでダウンロードが完了です

## 動画の検索

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481c8b036f2317cb8d0bb

動画を検索するには、動画のタイトル/説明、動画タグ、タイムタグを入力してください。メンバー名で検索すると、メンバーのプロフィールページに移動します。

### Web版

1. 画面右上の検索ボタンをクリック
2. 検索ワードを入力
3. 検索結果が表示されます（動画タグとタイムタグは別々に表示）

### アプリ版

1. 検索ボタンをタップ
2. 検索ワードを入力
3. 表示された候補から該当するものを選択
4. 検索結果が動画タグとタイムタグに分かれて表示されます

**※動画にタグを付けることで、より効率的に検索できます！**

## 動画タグをつける

**ヘルプセンターURL**: https://splyza.notion.site/176df661219481fa8cffe4171739b977

「動画タグ」は動画自体につけるタグです。動画の中のシーンにつけるタグとは別のものです。

### Web版

1. 動画を選択
2. 「編集」をクリック
3. 「動画タグ」の「＋」アイコンをクリック
4. 動画に関連するメンバーを選択
5. 「それ以外」をクリック
6. 必要なタグを選択（④をクリックで新しいタグを追加できます）
7. 「閉じる」をクリック

### アプリ版

1. 動画を選択
2. 動画タイトル下の「動画タグを追加(+)」をタップ
3. 「メンバー」から動画に関連するメンバーを選択
4. 「その他」をタップ
5. 必要なタグを選択
6. 「完了」をタップ

**新しいタグを追加する場合**

右上の「...」から「タグ管理」を選択し、新しいタグを作成してください。
